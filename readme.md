# 🍄 Super Mario All-Stars - Web Arcade

Edición especial para navegador basada en el clásico **Super Mario All-Stars** de Super Famicom. Ejecución directa, sin instalaciones y optimizada para jugar en cualquier dispositivo (PC y Móvil).

Este proyecto forma parte de la **Zona Arcade** de [Nautilus Lab](https://nautiluslaarhc.blogspot.com/), brindando acceso rápido a clásicos del entretenimiento digital.

## 🚀 Características

* **Emulación Nativa:** Corriendo sobre el núcleo `snes` de alto rendimiento.
* **Plug & Play:** Soporte para controles USB, Bluetooth y teclado.
* **Mobile-Friendly:** Controles táctiles responsivos integrados automáticamente.
* **Optimizado para Web:** Carga ligera y rápida a través de GitHub Pages.

## 🎮 Controles (PC)

| Acción | Tecla (PC) |
| :--- | :--- |
| **Movimiento** | `Flechas de Dirección` |
| **Saltar / Aceptar** | `Z` |
| **Correr / Cancelar** | `X` |
| **Start** | `Enter` |
| **Select** | `Shift Derecho` |
